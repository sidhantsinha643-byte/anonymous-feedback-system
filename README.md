# Anonymous Feedback System

## 📌 Description
The Anonymous Feedback System is a web-based platform where students can submit feedback about courses, faculty, and campus facilities anonymously.

## 🚀 Features
- User Registration & Login
- Anonymous Feedback Submission
- View Feedback
- Responsive Design (Dark/Light Mode)

## 🛠️ Prerequisites
- Python 3.x
- Flask
- MySQL / SQLite
- Web Browser
